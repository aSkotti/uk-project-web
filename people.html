Current Members

Jan Vitek
Professor

PI. Works on the design and implementation of programming languages.

Robert Grimm
Researcher

Studies the application of data analysis to questions of accountability and transparency. Current work involves statistical analysis of large public datasets to assess the effectiveness of regulatory frameworks.

Jakob Hain
PhD Student

Works on typed intermediate representations for dynamic languages, focusing on how static type information can enable compiler optimizations such as specialization, devirtualization, and inlining in R.

Joel Jakubovic
Postdoctoral Researcher

Studies programming systems and end-user programming, exploring how novel interaction paradigms and computational substrates can make programming more accessible to non-experts.

Matěj Kocourek
PhD Student

Works on JIT compilation techniques for R, investigating how copy-and-patch compilation can be used to build fast baseline compilers by reusing existing interpreter implementations.

Jan Kofroň
Professor

Works on software verification and program analysis, including model checking and static analysis techniques for improving software reliability.

Mickael Laurent
Postdoctoral Researcher

Works on set-theoretic types and type inference for dynamic languages. Current work includes algorithms for subtyping and tallying with union, intersection, and negation types, and their application to typing R programs.

Lucie Lerch
Administrator

Project coordination and administrative support.

Pavel Parízek
Professor

Works on program analysis and verification tools, including data lineage tracking for enterprise applications and debugging techniques that scale to large multi-threaded systems.

Tomáš Petříček
Professor

Studies how people program and the limits of making programming easy. Current work explores document-oriented end-user programming and interdisciplinary perspectives on the practice of data science.

Anastasija Skotorenko
Research Assistant

Alek Boruch-Gruszecki
